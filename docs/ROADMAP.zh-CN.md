# 路线图

**语言：** [English](ROADMAP.md) | 中文

本路线图描述 `wechat-gateway` 公开仓库的当前状态，并区分已经可用的内容和每个使用者需要在自己环境里补完的内容。

## 已经足够使用

- 本地网关服务
- 会话/状态存储模式
- 配置和集成 API
- Docker 部署示例

## 需要本地补全

- 共享或公开部署前强化鉴权
- 如有需要，增加加密状态存储
- 在公开仓库外增加特定集成适配器
- 补充状态文件备份/恢复运维说明

## 建议改进

- 把大型 server 拆分为 route/service 模块
- 补充 API 请求/响应示例
- 增加 mocked 下游服务的集成测试
- 为管理 UI 增加密钥处理保护

## 仍值得补充的文档

- 使用非私有演示数据制作截图或短录屏。
- 为常见请求/响应补完整 API 示例页。
- 为持久化数据路径补备份和恢复说明。
- 等公开用户开始部署后，基于真实问题整理排障页。

## 维护说明

- 公开示例保持通用。
- 英文和中文说明保持同步。
- 尽量拆小 issue 和 pull request，方便 AI 辅助贡献者安全接手。
- 发布新版本前重新跑敏感信息扫描。
