# 截图和演示素材

**语言：** [English](SCREENSHOTS.md) | 中文

公开仓库目前不提交私人部署截图。建议把可公开截图放到 `docs/screenshots/`，并只使用合成数据、公开演示数据或已打码内容。

## 建议补充的截图

- 使用演示状态的本地 UI 状态页
- 使用占位 URL 的集成设置页
- 使用合成 targetId 的 webhook 测试结果

## 添加截图前检查

- 不包含真实姓名、聊天记录、媒体标题、健康数据、订单信息、cookie、token、内网地址或生产 URL。
- 图片文件名清晰，例如 `overview.png`、`settings.png`、`report-demo.png`。
- 在 `README.zh-CN.md` 或本文中引用截图前，确认路径大小写正确。
- 截图更新时同步检查英文 `docs/SCREENSHOTS.md`。

## 推荐目录

```text
docs/screenshots/
  overview.png
  settings.png
  workflow.png
```
